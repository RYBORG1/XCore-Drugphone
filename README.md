Opsætning

1. Omdøb resource-mappen til xcore-drugphone hvis den ikke allerede hedder det.
2. Placér xcore-drugphone i din resources mappe.
3. Importér SQL filen install/xcore-drugphone.sql i din database.
4. Tilføj items fra install/items.lua til dine ox_inventory items.
5. Sørg for at disse resources er startet før xcore-drugphone:
qb-core
oxmysql
ox_lib
ox_inventory
ox_target
ps-dispatch
6. Tilføj ensure xcore-drugphone i server.cfg.
7. Restart serveren eller resource'en.
